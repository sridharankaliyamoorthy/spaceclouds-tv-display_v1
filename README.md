# Space Clouds TV Display Signage

This repository hosts the digital signage screens for Space Clouds, to be used with the PosterBooking platform on LG TVs.

## Directory Structure

- `/screens/left/`: Signature and Classic Cocktails.
- `/screens/center/`: Brand logo and promotional content.
- `/screens/right/`: Premium Spirits and Bottle Service.

## How to use with PosterBooking

1. Click on any image in the `/screens/` folders.
2. Click the **"Raw"** button to get the direct URL.
3. Use this URL in the PosterBooking **Dashboard > Media > Add Media** section.
